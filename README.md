# Santhosh.V | Fullstack Developer Portfolio 🚀

Welcome to the repository for my personal developer portfolio! This project serves as a digital representation of my skills, experiences, and the creative projects I've built. 

## 🌟 Overview

This portfolio is a Single Page Application (SPA) built using **React.js**. 
The primary goal of this platform is to showcase my journey as a Full Stack Web Developer and provide visitors with an intuitive, seamless browsing experience.

## 🛠️ Technology Stack

* **Frontend**: React.js
* **Styling**: Vanilla CSS3 (Custom Design System & Media Queries)
* **Animations**: AOS (Animate On Scroll)
* **Hosting/Deployment**: GitHub Pages (via `gh-pages` branch)

## 📁 Project Structure

* `components/` - All React UI modules (Navbar, Header, About, Skills, Experience, Projects).
* `assests/` - Contains all static media (images, SVGs, PDF resume file).
* `App.js` & `index.js` - Application entry points and central layout management.

## 🚀 Getting Started

To run this project perfectly on your local machine:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Santhosh-v11/Portfolio.git
   ```

2. **Navigate to the directory:**
   ```bash
   cd Portfolio
   ```

3. **Install Dependencies:**
   *(Note: This project uses `--legacy-peer-deps` due to specific styling/cursor libraries.)*
   ```bash
   npm install --legacy-peer-deps
   ```

4. **Start the Development Server:**
   ```bash
   npm start
   ```
   *The application will boot up at `http://localhost:3000`.*
